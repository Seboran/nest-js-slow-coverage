## Description

This folder is a [Nest](https://github.com/nestjs/nest) TypeScript repository containing the backend.
It will be useful to handle the requests and interact with the Models

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
